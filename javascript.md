# Javascript Fundamentals Review

## Primative Data Types

Javascript has several basic data types:

```javascript
'this is a string' // string (text)
123 // number (int)
1.2 // number (float)
true // boolean
false // boolean
[1,2,3] // object (array)
{
	prop1: 'value1',
	prop2: 'value1'
} // object
null // object (null)
undefined // undefined
```
